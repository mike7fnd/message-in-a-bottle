# **App Name**: Message in a Bottle (MiTB)

## Core Features:

- Anonymous Message Submission: Allow users to compose and send anonymous messages to others.
- Recipient Search: Implement a search functionality for users to find recipients by name. Provide alternative approach such as providing recipient link for extra security.
- Message Display: Display sent and received messages in a clean, readable format with timestamp.
- Generative AI Enhanced Message Polishing: Give the user the option to utilize a generative AI tool to rewrite and improve the composed message for tone, clarity, and emotional impact.

## Style Guidelines:

- Primary color: Black (#000000).
- Background color: White (#FFFFFF).
- Accent color: Light Gray (#D3D3D3).
- Font: 'Helvetica Neue Light' (sans-serif) as requested.
- Note: currently only Google Fonts are supported.
- Use minimalist icons. Replace default arrow by a bottle icon to signify returning back.
- Mobile-first design with responsive elements for desktop support. Centered content for easy readability on smaller screens. Consistent spacing and padding.
- Smooth transitions between screens and gentle animations for submitting/receiving messages to simulate floating in water.
